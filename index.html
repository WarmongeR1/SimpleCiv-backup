SimpleCiv

по-русски

SimpleCiv is really simple clone of Civilization game. It's free software (released under GNU GPL v2 license), written in pure C, using ncurses and expat libraries.

Downloads

  • Source code: zip, tar.gz or just use git clone git@github.com:Melkogotto/SimpleCiv.git
  • Linux binaries: tar.gz
  • Windows binaries: zip, tar.gz

Note: all binaries for i686 arch.

Note for Linux-users: you will need to install libncurses and libexpat packages.

If you found a bug…

…email me at melkogotto@gmail.com. Please write about it in details.

Thanks for any help in advance!

Features

  • technology tree and researchs, as in original Civilization, but with several exception: you don't know, what the technologies are in the end of this tree; you can't research some technology, if you don't have special nature resource.
  • hiring units and fighting between them
  • creating and capturing cities
  • multiplayer («hot seat»)

Or, in the nutshell, differents from Civilization are:

  • no buildings
  • no unit's experience or city's levels
  • no diplomacy
  • no AI
  • different technology tree implementation

SimpleCiv is really simple :)
